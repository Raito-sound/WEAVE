初回設定

Dropbox App Console で作成したアプリの App Key を入力してください。データはあなたの Dropbox(アプリ/WEAVE フォルダ)にのみ保存されます。

アプリ登録がまだの場合

  1. dropbox.com/developers → App Console → Create app
  2. Scoped access → App folder → 名前「WEAVE」等
  3. Permissions タブ: files.content.read / files.content.write にチェック → Submit
  4. Settings タブ: Redirect URIs に下記を追加
